# Story 16.2 — Estrutura de cursos, modulos e aulas

| Campo | Valor |
|--------|--------|
| **Epic** | 16 — Cursos |
| **Prioridade** | Alta |
| **Depende de** | 16.1 |
| **Desbloqueia** | 16.3, 16.4 |

---

## User story

**Como** escola,  
**quero** estruturar meus cursos em modulos e aulas,  
**para** organizar o conteudo de forma progressiva e reutilizavel.

---

## Contexto

- O curso precisa ter estrutura pedagogica clara.
- Aulas podem conter video, PDF, texto e recursos complementares.
- A ordem de modulos e aulas impacta o progresso do aluno.

---

## Comportamento esperado

1. Curso possui N modulos.
2. Cada modulo possui N aulas ordenadas.
3. Aula aceita video, PDF, texto e quiz associado.
4. Admin reordena modulos e aulas sem perder progresso historico.

---

## Criterios de aceite

1. Admin cria modulo dentro do curso.
2. Admin cria aula com ao menos um tipo de conteudo.
3. Ordem de exibicao de modulos e aulas e persistida.
4. Usuario visualiza o curso respeitando a sequencia definida.

---

## Fora do escopo

- Editor de video nativo.
- Streaming ao vivo.
- Conteudo SCORM/LTI.

---

## Definition of Done

- [x] Estrutura pedagogica documentada.
- [x] Regras de ordenacao e publicacao descritas.
- [x] Tipos de conteudo MVP definidos.
