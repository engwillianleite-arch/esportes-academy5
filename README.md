# Esportes Academy — UI Prototypes

Protótipos HTML das telas da plataforma **Esportes Academy**, gerados pelo squad de desenvolvimento.

## Convenção de nomenclatura

Todas as telas seguem o padrão `Portal-Tela`:

| Portal | Descrição |
|---|---|
| `SuperAdmin` | Gestão da plataforma SaaS (todos os tenants) |
| `Admin` | Painel do administrador da escola |
| `Escola` | Gestão no contexto da escola |
| `Professor` | Portal do professor |
| `Pais` | Portal dos pais/responsáveis |

## Telas geradas

### SuperAdmin
- `SuperAdmin-dashboard.html` — Visão geral da plataforma
- `SuperAdmin-schools.html` — Gestão de escolas (tenants)
- `SuperAdmin-usuarios.html` — Gestão de usuários
- `SuperAdmin-cobrancas.html` — Cobranças da plataforma
- `SuperAdmin-planos.html` — Planos & Licenças
- `SuperAdmin-notificacoes.html` — Notificações
- `SuperAdmin-configuracoes.html` — Configurações do sistema
- `SuperAdmin-permissoes.html` — Gestão de permissões por perfil

### Admin Escola
- `Admin-cobrancas.html` — Cobranças da escola

### Escola
- `Escola-grupos.html` — Gestão de turmas/grupos

## Stack

- HTML5 + CSS3 + JavaScript vanilla
- Chart.js para gráficos
- Inter (Google Fonts)
- Design system próprio com CSS custom properties

## Como visualizar

Abra qualquer arquivo `.html` diretamente no navegador. Não requer servidor.
